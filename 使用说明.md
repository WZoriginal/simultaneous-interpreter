# 使用说明

## 1. 启动

在 `E:\register\register4` 目录运行：

```powershell
npm start
```

浏览器打开：

```text
http://localhost:3000
```

## 2. 填写模型配置

右侧填写：

- API Key：你的 DashScope API Key
- Base URL：`https://dashscope.aliyuncs.com/compatible-mode/v1`
- 翻译模型：`deepseek-v4-flash`
- 电脑声音识别模型：`qwen3-asr-flash-realtime`

## 3. 识别电脑视频声音

1. 输入源选择 `电脑视频声音`。
2. 输入语言按视频语言选择，例如英文视频选 `English`。
3. 目标语言选择要翻译成的语言。
4. 点击 `开始传译`。
5. 在浏览器弹出的共享窗口里，选择播放视频的标签页、窗口或屏幕。
6. 一定勾选 `共享音频` 或 `共享标签页音频`。

## 4. 调整同传效果

- 想更快：把 `无标点分段延迟` 调低，比如 `300-500ms`。
- 想原文更短：把 `原文分句长度` 调低，比如 `50-70字符`。
- 想句子更完整：把 `原文分句长度` 调高，比如 `100-140字符`。

页面左边显示原文，右边显示对应译文，会自动滚动到最新内容。

## 5. 常见问题

没有声音识别：重新开始，选择视频来源时勾选共享音频。

识别语言不准：检查输入语言是否和视频语言一致。

接口错误：检查 API Key、Base URL、模型名；服务错误日志在 `server.log`。

导出记录：点击页面里的 `导出文本`。
