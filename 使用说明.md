# 快速使用指南

## 1. 本地启动服务

1. 在项目根目录下打开终端，安装项目依赖（仅在首次运行项目时需要）：
   ```bash
   npm install
   ```
2. 运行启动命令：
   ```bash
   npm start
   ```
3. 启动完成后，在浏览器中打开以下地址：
   ```text
   http://localhost:5423
   ```

---

## 2. 配置模型与 Key

在右侧 **配置面板** 中填写对应的配置：

- **翻译 API Key**：填入您的翻译服务密钥（例如 DeepSeek 官方 Key 或阿里云百炼 Key）。
- **翻译 Base URL**：如果使用的是 DeepSeek 官方，请填入 `https://api.deepseek.com/v1`；如果使用的是阿里云百炼，请填入 `https://dashscope.aliyuncs.com/compatible-mode/v1`。
- **翻译模型**：根据 API 服务商填写，例如 `deepseek-chat` 或 `deepseek-v4-flash`。
- **声音识别 API Key**：**如果您使用的是非阿里云 Key（如 DeepSeek）进行翻译**，则此处必须单独填入您的 **阿里云百炼 API Key** 以便进行语音识别；如果您翻译也是用阿里云 Key，此处**留空**即可，系统会自动复用上面的 Key。
- **识别模型 & WebSocket URL**：直接在下拉菜单中选择您想使用的语音模型（推荐选择 `qwen3-asr-flash-realtime` 或 `paraformer-realtime-v2`），**网页会自动为您绑定最匹配的后端接口 URL**，无需手动调整。

---

## 3. 音频捕获与翻译

- **麦克风模式**：适用于直接对着电脑讲话。输入源选择 `麦克风`，点击 `开始传译` 并允许浏览器调用麦克风即可。
- **电脑视频声音模式**：适用于翻译正在播放的网课、视频或网页会议。
  1. 输入源选择 `电脑视频声音`。
  2. 根据视频语言选对“输入语言”（例如英文视频选 `English`）。
  3. 点击 `开始传译`。
  4. **⚠️ 重要**：在浏览器弹出的屏幕共享窗口中，选择您播放视频的那个“标签页”或“窗口”，并且**一定要勾选弹窗底部的“共享音频”复选框**，否则系统无法捕获声音。

---

## 4. 常见微调技巧

在右侧面板下方，可以通过滑块微调传译效果：
- **想要翻译反应更灵敏**：将 `无标点分段延迟` 调低（如 `300-500ms`），将 `原文分句长度` 限制调小（如 `50-70` 字符），能显著减少翻译等待时间。
- **想要句子更完整、译文质量更高**：将 `原文分句长度` 限制调大（如 `100-140` 字符），这样模型能拿到更完整的上下文后再输出翻译。
