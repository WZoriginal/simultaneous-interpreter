# 同声传译工作台

本项目是一个本地网页工具，用于实时语音识别和大模型翻译。支持两种输入源：

- 麦克风：适合现场讲话、会议、课堂。
- 电脑视频声音：适合识别电脑里正在播放的视频、网页、播放器或会议声音。

翻译接口使用 OpenAI 兼容格式，当前默认配置为 DashScope：

```text
Base URL: https://dashscope.aliyuncs.com/compatible-mode/v1
翻译模型: deepseek-v4-flash
电脑声音识别模型: qwen3-asr-flash-realtime
```

## 启动方法

在项目目录运行：

```powershell
npm start
```

然后用浏览器打开：

```text
http://localhost:3000
```

如果端口被占用，可以设置其他端口：

```powershell
$env:PORT=3001
npm start
```

## 基本配置

打开页面后，在右侧“模型与语言”里填写：

1. `API Key`：DashScope 或其他 OpenAI 兼容服务的 Key。
2. `Base URL`：默认是 `https://dashscope.aliyuncs.com/compatible-mode/v1`。
3. `翻译模型`：默认是 `deepseek-v4-flash`。
4. `电脑声音识别模型`：默认是 `qwen3-asr-flash-realtime`。
5. `输入语言`：视频或讲话的原始语言。
6. `目标语言`：需要翻译成的语言。

API Key 只保存在当前页面输入框里，不会写入代码文件。Base URL、模型、语言、术语表和滑块设置会保存在浏览器本地存储。

## 麦克风模式

适用于直接对着电脑讲话。

1. 输入源选择 `麦克风`。
2. 选择输入语言和目标语言。
3. 填写 API Key。
4. 点击 `开始传译`。
5. 浏览器提示麦克风权限时，选择允许。

麦克风模式使用浏览器自带语音识别能力，建议使用 Chrome 或 Edge。

## 电脑视频声音模式

适用于识别电脑里正在播放的视频、网页、播放器或会议声音。

1. 输入源选择 `电脑视频声音`。
2. 如果视频是英文，输入语言选择 `English`；如果是中文，选择 `中文普通话`。
3. 填写 API Key。
4. 点击 `开始传译`。
5. 浏览器弹出共享选择器后，选择正在播放声音的标签页、窗口或屏幕。
6. 必须勾选 `共享音频` 或 `共享标签页音频`。

如果没有勾选共享音频，页面会提示没有捕获到电脑声音。

## 左右对照

页面左侧显示原文，右侧显示对应译文。每一段原文和译文按相同顺序对齐，方便做同声传译或字幕检查。

新增内容会自动滚动到底部，通常不需要手动拉滚动条。

## 延迟和分句长度

右侧有两个重要滑块：

- `无标点分段延迟`：默认 `500ms`。如果语音识别没有返回标点，等待这段时间后会把当前片段送去翻译。
- `原文分句长度`：默认 `90字符`。如果识别出的原文太长，会按这个长度提前切成更短片段。

建议：

- 更实时：把原文分句长度调到 `50-70`，延迟调到 `300-500ms`。
- 更完整：把原文分句长度调到 `100-140`，延迟调到 `600-900ms`。
- 英文演讲：通常 `70-110` 比较合适。
- 中文讲话：通常 `50-90` 比较合适。

## 术语表

可以在“术语表”里填写固定译法，例如：

```text
注册系统 = registration system
张三 = Zhang San
DashScope = DashScope
```

模型翻译时会优先参考这些译法。

## 导出记录

点击 `导出文本` 可以导出本次传译记录，格式包含原文和译文。

## 常见问题

### 点击开始后没有识别

- 麦克风模式：检查浏览器是否允许麦克风权限。
- 电脑视频声音模式：重新点击开始，选择播放视频的标签页，并勾选共享音频。
- 确认视频本身有声音，并且没有静音。

### 原文识别不准

- 检查输入语言是否选对。英文视频要选 `English`，中文视频要选 `中文普通话`。
- 电脑视频声音模式比“外放再用麦克风收音”更准，优先使用它。

### 翻译显示接口错误

- 检查 API Key 是否正确。
- 检查 Base URL 是否为 `https://dashscope.aliyuncs.com/compatible-mode/v1`。
- 检查模型名是否可用。当前后端对 DashScope 的 404 模型错误会自动尝试 `qwen-flash` 兜底。
- 如果服务崩溃，可查看项目目录里的 `server.log`。

### 视频播放时页面停止识别

通常是共享来源被关闭、浏览器停止了屏幕共享，或没有共享音频。重新点击开始并选择共享音频即可。

## 技术结构

```text
server.js          本地 HTTP 服务、翻译接口代理、DashScope 实时 ASR WebSocket 代理
public/index.html  页面结构
public/styles.css  页面样式
public/app.js      前端交互、音频采集、分句、左右对照显示
```

## 安全提示

不要把真实 API Key 提交到公开仓库。当前工具不会把 API Key 写入代码文件，但如果 Key 曾经发到聊天或日志里，建议在控制台重新生成一个新的 Key。
