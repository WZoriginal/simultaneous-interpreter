# 同声传译工作台

本项目是一个轻量、快速、开箱即用的本地网页端同声传译与实时语音转写工具。它支持将外界讲话或电脑系统音频转写为文字，并通过大语言模型进行实时对照翻译。

---

## 🌟 核心特性

- **双输入源支持**：
  - **麦克风**：适合现场演讲、会议、课堂教学的语音转写与翻译（使用浏览器内置 Web Speech API，完全免费且不消耗 ASR 额度）。
  - **电脑视频声音**：采用浏览器屏幕音频采集技术，直接提取系统/网页的原始声音输入。无需外放收音，音质无损，转写识别极度精准，特别适合网课、视频翻译及远程会议。
- **全新配置面板（更宽敞、更清爽）**：
  - **独立 API Key & Base URL**：翻译和语音识别（ASR）可以使用不同的云服务商或账号，例如使用 **DeepSeek 官方 Key** 进行高质量翻译，使用 **阿里云百炼 Key** 进行高性能语音转写。
  - **双重自适应绑定**：内置了阿里云目前最主流的 ASR 语音模型下拉菜单。选择 `qwen3-asr` 或 `paraformer` 系列模型时，前端与后端会自动识别并无缝切换对应的 **WebSocket 接入协议（Realtime 协议 vs Inference 协议）** 及 URL 端点，真正实现零配置上手。
- **左右分栏对照**：原文与译文段落精准按顺序水平对齐，新增内容自动滚动，非常适合充当实时字幕或同传对照检查。
- **播报与导出**：支持翻译结果自动语音播报（TTS），并可一键导出本次传译记录。

## 🚀 快速启动

1. 克隆或下载本项目到本地。
2. 在项目根目录下打开终端，安装依赖并启动服务：
   ```bash
   # 安装依赖（仅在首次启动时需要）
   npm install

   # 启动服务
   npm start
   ```
3. 服务启动后，在浏览器（推荐使用 Chrome 或 Edge）中打开：
   ```text
   http://localhost:5423
   ```

*提示：如果默认端口 `5423` 被占用，您可以通过环境变量指定其他端口启动，例如：*
*Windows PowerShell:*
```powershell
$env:PORT=3000
npm start
```
*Linux/macOS:*
```bash
PORT=3000 npm start
```

## ⚙️ 模型配置说明

打开工作台页面后，在右侧 **“配置面板”** 中进行设置：

### 1. 翻译引擎设置
- **翻译 API Key**：填写您的 API 密钥（例如 DeepSeek 官方 Key 或阿里云百炼 Key）。
- **翻译 Base URL**：填写符合 OpenAI 兼容标准的 API 接口地址。例如：
  - 阿里云百炼（默认）：`https://dashscope.aliyuncs.com/compatible-mode/v1`
  - DeepSeek 官方：`https://api.deepseek.com/v1`
- **翻译模型**：对应的模型名称。例如 `deepseek-v4-flash`（默认，极速且性价比高）或 `deepseek-chat`。

### 2. 语音识别设置 (ASR)
- **声音识别 API Key**：填写用于语音转写的 API 密钥（目前连接阿里云服务，需使用百炼 Key）。**如果留空，系统将自动回退使用上方填写的“翻译 API Key”**，免去重复输入的繁琐。
- **声音识别 Base URL**：选择您的 WebSocket 服务端点。系统已内置国内默认和海外端点，且会根据您选择的模型自适应切换 `/realtime` 或 `/inference` 路径。
- **识别模型**：下拉选择内置的阿里云百炼大模型：
  - `qwen3-asr-flash-realtime`：千问最新一代极速语音大模型（推荐）。
  - `paraformer-realtime-v2`：Paraformer 第二代通用实时识别模型（推荐，识别率极高）。
  - 也支持选择 `自定义模型名称...` 并手动键入其他模型 ID。

### 3. 传译偏好与参数微调
- **语言选择**：支持中文普通话、英语、日语、韩语、法语、德语、西班牙语等多语种实时互译。
- **译文语气**：例如“自然、清晰、适合会议口播”等，大模型会根据该指令调整翻译风格。
- **术语表**：支持指定专有名词的固定译法（例如：`注册系统 = registration system`），多行输入，翻译时大模型会优先参考。
- **无标点分段延迟**：如果语音识别长时间没有输出标点，等待指定毫秒数后，会自动将当前未完结的句子发送翻译。
- **原文分句长度**：当单句字数达到限制时，系统会自动切分发送给模型，以保证翻译的实时性。

## 📖 模式使用指引

### 麦克风模式（现场讲话）
1. 输入源选择 `麦克风`。
2. 填写 **翻译 API Key**。
3. 点击 **`开始传译`**。
4. 浏览器弹出权限申请时，选择“允许使用麦克风”。

### 电脑视频声音模式（视频/会议翻译）
1. 输入源选择 `电脑视频声音`。
2. 确保配置面板中的 **声音识别 API Key** 或 **翻译 API Key** 中填写了有效的**阿里云百炼 Key**。
3. 点击 **`开始传译`**。
4. 浏览器会弹出屏幕共享选择弹窗，请选择您**正在播放视频/会议音频的“标签页”或“窗口”**。
5. **⚠️ 关键：必须勾选下方的“共享音频”或“共享标签页音频”复选框**，否则系统无法捕获声音。

## 🛠️ 技术架构

项目结构极简，非常适合学习和二次开发：
- [`server.js`](file:///e:/register/register4/server.js)：后端 Node.js 服务。主要负责本地 Web 托管、大模型翻译接口反向代理，以及双协议自适应的 DashScope ASR WebSocket 双工转发。
- [`public/index.html`](file:///e:/register/register4/public/index.html)：纯 HTML5 前端布局与交互面板。
- [`public/styles.css`](file:///e:/register/register4/public/styles.css)：极简而 premium 的现代 CSS 样式设计。
- [`public/app.js`](file:///e:/register/register4/public/app.js)：前端核心逻辑。包含 Web Audio API 音频采集、分句控制、本地缓存记忆、TTS 语音播报及左右对照高度同步计算。
